Лингвистический анализ хоррор-подкаста "The Magnus Archives"

Данные: https://drive.google.com/drive/folders/1PjErmu9w9cm49r5XOhMGQgqwdmNujoZB?usp=sharing

Цель: Провести комплексный количественный анализ транскриптов подкаста для выявления динамики лексических паттернов, связанных со Страхами (The Fears), языка от сезона к сезону и динамики значимости персонажей. 

Задачи проекта:

  1. Обработать данные транскриптов из файлов .txt в единый словарь, где сезоны соотносятся с эпизодами, а названия эпизодов – с текстами этих эпизодов. 
  
  2. Провести TF-IDF анализ для определения сигнатурных слов, характерных для каждого сезона, и проследить эволюцию лексики от сезона к сезону.
  
  3. Создать словари ключевых слов для 14 Сущностей Страха и построить тепловую карту их распределения по сезонам для проверки гипотезы о сезонном доминировании определённых Страхов.
  
  4. Визуализировать полученные результаты в виде набора графиков и диаграмм.

Логическая структура проекта:

  1. Загрузка и парсинг данных — извлечение текстов эпизодов из сырых транскриптов.
  
  2. Предобработка текста — токенизация, удаление стоп-слов, очистка от пунктуации.
  
  3. Лексический анализ — расчёт TF-IDF и построение облака слов.
  
  4. Тематический анализ — картирование Сущностей Страха через ключевые слова.
  
  5. Нарративный анализ — отслеживание персонажей и их динамики.

Методология проекта:

  Парсинг данных – Python, регулярные выражения – Извлечение текстов эпизодов из структурированных файлов
  Токенизация и очистка – NLTK, стоп-слова английского языка – Разбивка текста на слова, удаление шума
  TF-IDF анализ – Scikit-learn (TfidfVectorizer) – Выявление сигнатурных слов для каждого сезона
  Анализ персонажей – Собственные функции – Отслеживание частоты упоминаний
  Лексическое картирование – Словари ключевых слов, взвешенное суммирование – Количественная оценка присутствия Сущностей Страха
  Визуализация – Matplotlib, WordCloud – Построение тепловых карт и графиков
